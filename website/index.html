Security Notes

上线前建议先处理安全配置

  • 把数据库密码、邮箱授权码移到环境变量。
  • 上线使用 HTTPS，关闭 Android 明文 HTTP。
  • 给 API 增加登录鉴权，避免只依赖 userId 和 coupleId。
